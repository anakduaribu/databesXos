CARA PENGGUNAAN PROGRAM:

1. Pastikan perintah make sudah terinstall dan dapat dijalankan

2. Masuk ke dalam root direktori dari file program

3. jika sudah terdapat file bin dan obj, jalankan perintah make clean, untuk menghapus kedua folder tersebut

4. jika thread pada komputer tidak cukup, turunkan thread yang ada di file txn/txn_processor_test.cc di bagian CPU_SET pada program utama

5. jalankan dengan menggunakan command make test pada root direktori

6. tunggu hingga semua transaction selesai diproses menggunakan ketiga algoritma yaitu simple locking exclusive only, Serial Optimistic Concurrency Control dan Multiversion Timestamp Ordering Concurrency Control
